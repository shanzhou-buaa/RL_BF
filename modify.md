# 强化学习的环境设置
该部分主要是根据Liu2020文章中公式25修改

# 状态空间
- 通信信道矩阵 `H`：代码中形状为 `K x M`，实部+虚部，总维度 `2*K*M`。


# 动作空间
- 当前联合波束矩阵 `W=[Wc, Wr]`：`Wc` 为 `M x K`，`Wr` 为 `M x M`，实部+虚部，总维度 `2*M*(K+M)`。

每次优化不要输出动作的delta修正值，而是直接优化整个动作W。

# 奖励函数
记各通信用户 SINR：$\gamma_k$，SINR的阈值为$\Gamma$

记完整雷达损失$ L_r = L_{r,1} + w_c*L_{r,2}$，在实际实验中设置$w_c=1$。

奖励函数$r=-L_r-\Sigma_{k=1}^K R_{0,k}-\|diag(R)-P_t/M\|_2^2$

奖励函数中第一项是雷达的指标，由于强化学习优化时是让reward越大越好，因此设置为负号，即要求雷达的loss越小越好

奖励函数中第二项含义是 if $\gamma_k<\Gamma, R_{0,k}=R_{SINR}$，else $R_{0,k}=0$，$R_{SINR}$是一个较大的数，即当通信用户的SINR不满足阈值时给一个惩罚

奖励函数中第三项含义是要求功率约束，只看矩阵的对角线元素。

以上是我倾向的设定，另外还要考虑公式中的$\alpha$是否要加入动作空间（即待优化的变量）中

除此之外，理想的效果应该是改进的PPO比原始的PPO，波束图在主瓣增益更好，而旁瓣增益更低（体现优化项中的$L_{r,1}$）；除此之外优化项中的$L_{r,2}$可以在论文中通过修改SINR给出整个$L_{r}$的变化曲线来体现，所以对二者的优化都是可以体现的，波束图虽然是更重要的，但不应该对$L_{r,1}$进行针对性的调整，它更优应该是建立在整个$L_{r}$都更优的基础上的。
